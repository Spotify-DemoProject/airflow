# airflow
Spotify 데이터 파이프라인을 스케줄링합니다.
Docker 컨테이너를 통해 서비스 오케스트레이션을 구성하고 있습니다.<br><br>

https://docs.docker.com/engine/install/ubuntu/
2. run commands to setup
```
sudo docker compose up -d
```
